# Diabetes Prediction using Machine Learning

This project is a simple machine learning based system to predict diabetes using
a Support Vector Machine (SVM) model. The project focuses on basic data preprocessing,
feature scaling, and model evaluation to understand how machine learning can be applied
to medical datasets for prediction tasks.
